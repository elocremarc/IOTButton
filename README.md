# Simple IOT Button

Barebones Raspbery Pi IOT ON/OFF button. This runs a simple web page on a webserver like Nginx/Appache the styling is based on Bootstrap 4. IT usees JQUERY/AJAX to dynamically open a PHP page which runs a simple GPIO commands from the terminal using the Wiring Pi library.
